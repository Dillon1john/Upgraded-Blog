# Upgraded-Blog
Created a multipage blog using the flask web framework. Site also includes a contact page that takes form and submits to my email.  python program utilizes modules, decorators, and multiple functions that redirect to different pages of the site and performs different actions. HTML templates also utilizes bootstrap framework.
